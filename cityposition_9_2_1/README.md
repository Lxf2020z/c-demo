
## 全国省市区经纬度查看器

### 目录结构描述
├── Readme.md                   // help    
├── cityposition_9_2_1.pro      // 配置    
├── widget.h                    //头文件    
├── main.cpp                    //主函数     
├── widget.cpp                  // 资源文件     
└── widget.ui                   //页面文件     

### 主要功能
1、运行程序，弹出显示地区经纬度的弹框，默认是北京市天安门的经纬度    
2、点击 省份组合选框 的下拉键可选择省份    
3、点击 市县组合选框 的下拉键可选择已选中省份的市县    

### 部署步骤
Qt Creator 环境下直接运行
